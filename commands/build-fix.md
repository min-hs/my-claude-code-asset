# Build Fix 커맨드

빌드 에러를 분석하고 수정합니다.

## 실행 순서

1. 빌드 에러 확인
```bash
make clean && make 2>&1
```

2. 에러 메시지 분석
- 컴파일 에러: 문법, 타입, 선언 누락
- 링크 에러: 심볼 미정의, 라이브러리 누락
- 헤더 에러: include 경로, 순환 참조

3. 수정 후 재빌드
```bash
make
```

4. 정적 분석 확인
```bash
cppcheck --enable=all src/
```

## 주의사항
- 한 번에 하나의 에러만 수정
- 수정 후 반드시 재빌드로 확인
- 워닝도 가능하면 제거
